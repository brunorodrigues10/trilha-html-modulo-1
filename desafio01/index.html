Tags: Título

As tags de título são usadas para definir títulos e subtítulos em uma página HTML. Existem seis níveis, que vão de h1 (mais importante) até h6 (menos importante). Elas ajudam na hierarquia visual e na organização do conteúdo da página.

Tags: Marcação de texto

Essas tags são utilizadas para destacar partes do texto com diferentes significados ou estilos:
strong: dá ênfase forte, geralmente exibido em negrito.
em: dá ênfase moderada, geralmente exibido em itálico.
b: deixa o texto em negrito, mas sem significado semântico.
i: deixa o texto em itálico, também sem semântica.
u: sublinha o texto.
mark: destaca o texto como se estivesse com marca-texto.
small: exibe o texto em tamanho menor, geralmente usado para observações.

Tags: Listas

O HTML oferece diferentes tipos de listas para organizar informações:
ul (lista não ordenada): apresenta os itens com marcadores (bolinhas).
ol (lista ordenada): apresenta os itens com numeração.
li: representa cada item dentro de uma lista (usado tanto em ul quanto em ol).
dl (lista de definição): usada para listas de termos e definições.
dt define o termo.
dd define a descrição do termo.
